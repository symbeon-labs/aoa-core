# 🧩 AOA Pattern Library: Assets

Esta biblioteca contém as matrizes geométricas fundamentais (Blueprints) para a geração de Assinaturas Ópticas Atômicas.

## 📁 Ativos Disponíveis

| Arquivo | Descrição | Uso |
| :--- | :--- | :--- |
| `aoa_credential_marker.svg` | Padrão concêntrico para credenciais de acesso fixas. | Docas, Pedágios, Portões. |
| `aoa_identity_seed.svg` | Matriz base para geração de identidades dinâmicas. | Ativos móveis, Smartphones. |
| `aoa_vehicle_atomic.svg` | **O Padrão Mestre do GuardTag**. | Para-brisas, Identificação Veicular. |


## ⚙️ Especificações de Impressão

Todos os ativos devem ser processados utilizando a especificação de **Micro-Geometria DTM** descrita em `/spec/GEOMETRY_V1.md`.

- **Resolução mínima:** 2400 DPI.
- **Camadas:** Requer sobreposição espectral (Visible + IR).
